User Registartion with Login/Logout

User registation form web application is implemented using MEAN stack. Angular is serving as the frontend and the back is handled by Node, Express and Mongo. The user registers via email & password which are saved in database but password is first encrypted using bcrypt and salt-secret. Also served side email validation is done. The web application also has the feature of login/logout. Angular frontend implements basic form validations. On login the user is taken to his/her profile page authentication via JWT and application implements prohibition to illegal routes.
